Testing Instructions for NSS Team:

  • Start at Step 1: Begin in your normal ConnieRTC Flex environment
  • Navigate to Step 2: Click the "Admin Tools & Data" button in the main interface
  • Choose Your Path:
    • Path A (Green): Test Channel Manager features - view NSS phone numbers and configurations
    • Path B (Red): Test Data Center features - access analytics and playback tools
  • Test All Features: From the Data Center Hub, test all three sections:
    • Voicemail Playback (Purple) - Listen to recent voicemail messages
    • Fax Viewer (Purple) - View and download fax documents
    • Analytics Dashboard (Purple) - Review live contact center performance data
  • Report Issues: Document any errors, slow loading, or unexpected behavior at each step
