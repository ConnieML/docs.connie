# ConnieRTC Flow Diagrams

This directory contains user flow diagrams and navigation maps for the Connie platform.

## Available Diagrams

### NSS Admin Tools Flow (`nss-admin-tools-flow.mmd`)
- **Purpose**: Visual navigation guide for NSS UAT testers
- **Created**: August 12, 2025
- **Format**: Mermaid diagram
- **Usage**: Email communications, training materials, documentation

## How to Use Mermaid Diagrams

### Option 1: Mermaid Live Editor (Recommended)
1. Go to https://mermaid.live
2. Copy the contents of the `.mmd` file
3. Paste into the editor
4. Export as PNG or SVG

### Option 2: GitHub/GitLab
- These platforms automatically render `.mmd` files in markdown

### Option 3: VS Code Extension
- Install "Mermaid Preview" extension
- Right-click on `.mmd` file â†’ "Open Preview"

## Diagram Conventions

- **Entry Points**: Blue boxes with thick borders
- **Channel Management**: Green path
- **Data/Analytics**: Orange path  
- **Coming Soon**: Gray dashed boxes
- **Icons**: Used to quickly identify function types
